# 🏪 Store Sales Analysis Project | Power BI

 ## 📌 Project Overview
 This repository contains an interactive **Power BI Store Sales Analysis Dashboard** developed to analyze and visualize sales performance across different business dimensions. The project helps in understanding customer behavior, sales trends, delivery performance, and regional sales distribution through interactive visualizations and business insights. 

 The Power BI report consists of **two interactive tabs** designed for detailed sales analysis and business decision-making.

 ---

 ## 🚀 Project Features 
- Interactive Power BI dashboard
- Dynamic filtering and drill-through analysis
- Tracking Overall Sales performance
- Analysis Order Quantity sold
- Understand customer demographics
- Track delivery and channel performance
- Gender & Age Group Analysis
- Ship-city and Ship-state sales visualization

---

# 📈 Dashboard : Sales Analysis 
This dashboard provides a complete overview of sales performance using various visual reports.

## 🔹 Key Insights Included 
### ✅  📈Category-wise Sales Analysis
Analyze total sales generated by different product categories to identify top-performing categories. 

### ✅ 📅Size-wise Sales Analysis
Visualize sales distribution based on product sizes. 

### ✅ 🏪Channel-wise Sales Analysis 
Compare sales across different sales channels/platforms.

### ✅ 📑Delivery Status Analysis 
Track order delivery performance and order statuses.

### ✅ 📊Gender-wise Sales Analysis 
Understand purchasing behavior based on customer gender. 

### ✅ 🔝Top 10 Ship State & Ship City Analysis
Analyze sales distribution across different states and cities. 

### ✅ 📬Quarterly Sales Analysis 
Monitor sales trends and business growth quarter-wise.

### ✅ 🔍Age Category Sales Analysis 
Identify customer purchasing patterns based on different age groups( Younger Age, Mid Age, Older Age).

The dashboard allows users to interact with charts and filters for deeper analysis and better understanding of customer behavior and sales performance.
